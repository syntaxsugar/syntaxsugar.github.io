How to disable keyboard shortcut for Lockscreen

Date Sun, 30 Mar 2014 Tags Ubuntu / Gnome

System Tools > System Settings > Keyboard > Shortcuts > System > Lock Screen

Works well on Ubuntu 12.04 (Precise Pangolin) with GNOME Session Fallback.
