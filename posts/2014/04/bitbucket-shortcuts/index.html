Bitbucket Shortcuts

Sat, 26 Apr 2014
By Jaromir Fojtu

Here are some of the shortcuts I try to use often on Bitbucket to help reduce the amount of the time I am on there.

Useful Shortcuts

  • ? - To view help menu for shortcuts

Repository Navigation

  • r i - To view Issues
  • r s - To view Source Code
  • r c - To view Commits
  • r o - To view Overview
  • r d - To view Downloads

Issues

  • x i - Create Issue

Vimium Exception

If you are using Chrome plugin Vimium so you must add exception for Bitbucket

This will disable Vimium on Bitbucket:

http*://bitbucket.org/*
