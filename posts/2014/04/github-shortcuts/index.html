Github Shortcuts

16 Apr 2014

Here are some of the shortcuts I try to use often on Github to help reduce the amount of the time I am on there.

Useful Shortcuts

  • ? - To View help menu for shortcuts.
  • g c - To view the Code of the repo.
  • g p - To View Pull Requests.
  • g i - To view Issues.

This will disable Vimium on Github:
