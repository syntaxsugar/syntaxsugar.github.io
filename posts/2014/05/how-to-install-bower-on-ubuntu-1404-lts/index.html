How to install Bower on Ubuntu 14.04 LTS

Date Sun, 18 May 2014 Tags Ubuntu / Bower

Easy way

$ sudo apt-get install git-core
$ sudo apt-get install nodejs nodejs-legacy
$ sudo apt-get install npm
  • Install Bower
    $ sudo npm install -g bower

Reference

  • How To Install and Use Bower on your VPS
  • How To Install Node.js on an Ubuntu 14.04 server
  • error /usr/bin/env: node: No such file or directory Ubuntu specific error. We need to make link ln -s /usr/bin/nodejs /usr/bin/node or better use nodejs-legacy package:
    $ sudo apt-get install nodejs-legacy
